Abstract

TL;DR: Designed line following and path planning algorithms to navigate a robot around a track.


A task was set for the robot to make its way around the arena and avoiding obstacles. The using vision the robot was tasked with again making its way around the arena steering clear of obstacles while also detecting friendlies and enemies.


Parts

Infrared

Below shows the outcome of the visionless system. The Robot navigates its way around the arena using only the IR sensor array and the ultrasonic sensor at the front. In this implementation a Propeller P2X8C4M64P chip was used. The Propeller is a multicore chip and was used to control the line following algorithm, and the other peripherals such as the mini LCD, ultrasonic sensor and LED's.



Infrared + Vision

Below shows the outcome system with vision. The Robot navigates its way around the arena using the IR sensor array and the ultrasonic sensor, and the camera is used to detect friendlies or enemies around the arena. In this implementation a Raspberry Pi was used alongside the Propeller P2X8C4M64P. The Propeller controlled the line following algorithm, same as the infrared only implementation, and the RPi controlled the camera and the detection of blocks along the track.
